## How to Run

1. **Clone the repository:**

   git clone https://github.com/jahnviChauhan/Python-mini-projects.git

2. **Navigate to the project folder you're interested in:**

   cd "Main projects/<project-folder-name>"

3. **Run the Python script:**

   python3 main.py
